# 设计模式笔记

## 面向对象设计原则

1. 单一指责原则
2. 开闭原则
3. **里氏代换原则: 所有引用基类对象的地方能够透明使用其子类的对象。**
4. 依赖倒转原则
5. 接口隔离原则
6. 合成复用原则
7. **迪米特法则: 一个软件的实体应当尽可能少的和其他实体发生相互作用。**
